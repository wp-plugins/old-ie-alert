Display an upgrade nag on IE7 and lower.

== Description ==

Display an upgrade nag for people who are still using Internet Explorer 7 or lower. The nag also provides links to download Chrome, Firefox, Safari and Internet Explorer.

== Installation ==

Install, activate, done.

== Uninstall ==

Deactivate the plugin, delete if desired.
